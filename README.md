Biblioteca - Java POO Ej1

Gestión básica de una biblioteca con usuarios, trabajadores y préstamos de libros.

Proyecto de Java orientado a objetos que permite:
  - Registrar libros, usuarios y trabajadores.
  - Gestionar préstamos de libros.
  - Consultar libros prestados por usuarios o gestionados por trabajadores.
  - Enviar notificaciones a usuarios y trabajadores.

Se ha puesto en practica la programación orientada a objetos, empleando clases, herencia, interfaces y abstracción.
